New Music Friends

New Music Friends is a project I have been working to connect people through shared Musical Interests and allow them to discover new music through those connections. I got the idea from all the beautiful and amazing people I have met at the various concerts and festivals I have been to. These people where from all over the world and all unique and different, sometimes the only thing it seemed we would have in common is our music. This commonality allowed us to share the other parts of our lives and learn so much, and of course recommend each other new bands.

This project is still currently in it's prototype phase. Most of the core functionality is there, but there are many new features on their way. This project was also a learning one. I implemented it using the MERN stack (Mongo, Express, React, Node.js). I chose this stack because it seemed like one I could quickly get up and running, to test the idea out. I used bootstrap on the front end, along with custom SASS code to get the look I wanted out of it. It is deployed on Heroku, to allow for easy scaling and hassle free updates. I learned so much doing this project about the full development cycle of a web app, from the backend to the frontend to deployment. After this project is complete, I plan to take a step back and look for other technologies I can incorporate into my development strategies, to build off the foundation of knowledge I learned working on the project.

Flower Lad

Flower Lad was built for my class Game Design I. We were challenged to build a game that did not rely on violence to be compelling. I chose to create a game about a little flowerpot robot that enjoys poetry and planting flowers. I wanted to create and experience that was uplifting and relaxing, while at the same time being engaging to the player. You can download it yourself to see if i succeeded!
